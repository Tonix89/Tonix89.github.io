School Projects

RainyDays

RainyDays is an online store for all types of rain jackets. This cross-course project was develop througout our first year in front-end development course. First it was build using html and css in the first semester, then later added functionality using JavaScript in the second semester.

Community Science Museum

Community Science Museum is my first semester project in front-end development course at NorOff School Technology and Digital Media. Community Science Museum is a website that will encourage viewers to visit the museum physically. This website was only built using html and css.

Daily Devotion

Daily Devotion is a blog site. Where Christians all over the world can share their personal devotion to God and to inspire others. This is my project exam for my first year in front-end development course at NorOff School of Technology and Digital Media.
